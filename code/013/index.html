机动车辆保险条款示例

第一章 基本险

第一条

第一条第一款

机动车辆保险所承保的机动车辆是指汽车、电车、电瓶车、摩托车、拖拉机、各种专用机械车、特种车。

第一条第二款

本保险分为车辆损失险和第三者责任险，保险人按承保险别分别承担保险责任。

第二条

第二条第一款

1、发生保险事故时，被保险人对保险车辆采取施救、保护措施所支出的合理费用，保险人负责赔偿。但此项费用的最高赔偿金额以保险金额为限。

2、载运保险车辆的渡船遭受自然灾害(只限于有驾驶员随车照料者)。

3、第三者责任险的每次事故最高赔偿限额分为五个赔偿档次：5万元、10万元、20万元、50万元、100万元，被保险人可以自愿选择投保。

4、保险车辆发生第三者责任事故时，按《道路交通事故处理办法》(1991年9月22日中华人民共和国国务院令第89号)、有关法律、法规和保险合同的规定，在保险单载明的赔偿限额内核定赔偿数额。对被保险人自行承诺或支付的赔偿金额，保险人有权重新核定或拒绝赔偿。

第二章 管辖

第一条

第一条第一款

被保险人允许的合格驾驶员在使用保险车辆过程中发生意外事故，致使第三者遭受人身伤亡或财产的直接损毁，依法应当由被保险人支付的赔偿金额，保险人依照保险合同的规定给予赔偿。但因事故产生的善后工作，由被保险人负责处理。
车辆的保险价值根据新车购置价确定。车辆损失险的保险金额可以按投保时保险价值或实际价值确定，也可以由被保险人与保险人协商确定，但保险金额不得超过保险价值，超过部分无效。

第一条第二款

保险车辆全车被盗窃、被抢劫、被抢夺，以及在此期间受到损坏或车上零部件、附属设备丢失，以及第三者人员伤亡或财产损失。
